# 🌦 Weather Dashboard

A dynamic weather application that fetches real-time weather data using API integration.

---

## 🚀 Features

- Search weather by city name  
- Displays temperature in Celsius  
- Shows humidity levels  
- Displays wind speed  
- Weather condition with icons  
- Dynamic UI based on weather conditions  

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- OpenWeather API  

---

## 🔐 API Security

The API key is stored securely using a separate config file and is not exposed in the repository.
